Why Move Beyond Transformers?

Quadratic Attention Cost

O(n²) time and memory — 128K context is 16x more expensive than 32K. FlashAttention reduces memory but the compute cost remains.

KV Cache Growth

Llama 3 70B: ~1.25 MB per token in FP16. At 128K tokens the cache exceeds 160 GB — more than a single H100.

Fixed Computation Per Token

Every token gets identical FLOPs — "the" costs the same as a complex reasoning step. Uniform allocation is wasteful.

Inference Latency

Generation latency grows linearly with context length per decode step. Real-time streaming over long contexts becomes impractical.

State Space Models: Foundations

Continuous-Time Dynamical System

h'(t) = A h(t) + B x(t)
y(t)  = C h(t) + D x(t)

After discretization (step size Δ):
h_t = Ā h_{t-1} + B̄ x_t     ← recurrent mode
y   = conv(K, x)            ← convolutional mode

The HiPPO initialization for matrix A optimally compresses continuous signals into finite-dimensional state — recent inputs at higher fidelity.

S4 (Gu et al., 2021) achieved breakthrough results on Long Range Arena, solving Path-X (16K sequences) where dense attention failed.

SSM Dual Computation

  • Recurrent mode — O(1) per step, O(1) memory. No KV cache needed.
  • Convolutional mode — O(n log n) via FFT for parallel GPU training.
  • State vector: 16–256 dims per channel — compressed history of entire sequence.

From S4 to Mamba

Mamba: Selective State Spaces

  • Makes B, C, Δ input-dependent — content-based filtering
  • Hardware-aware parallel scan exploiting SRAM hierarchy
  • Trains 3–5x faster than equivalent transformers
  • At 1.4B: matched Pythia/GPT-Neo quality
  • 5x throughput at 1M token sequences

Mamba-2: SSD Framework

  • Revealed duality between selective SSMs and structured attention
  • SSM computation as matrix multiplications on tensor cores
  • 2–8x speedup over Mamba-1
  • Larger state dimension (64–256 vs 16)
  • Consistent quality match at 2.7B scale

Linear Attention Variants

Core Idea

Standard:  softmax(QK¹ / √d) V     → O(n²)
Linear:    φ(Q)(φ(K)¹ V)          → O(nd²)

By computing φ(K)¹V first (d×d matrix),
complexity becomes linear in sequence length.

Gated Linear Attention (GLA)

S_t = G_t ⊙ S_{t-1} + k_t v_t¹ — data-dependent gating solves unbounded state growth. Quality competitive with Mamba at 1.3–2.7B. Chunk-wise parallel training at 80–95% of FlashAttention-2 throughput.

Performer

FAVOR+ random feature maps approximate softmax. Linear complexity but 1–3 point perplexity gap that widens at scale.

cosFormer

Cosine-based reweighting with position decay. Gap narrowed to 0.5–1.0 perplexity points on WikiText-103.

TransNormerLLM

Lightning Attention + exponential decay + gating. Competitive perplexity at 1B–7B scale.

Lightning Attention & Limitations

Lightning Attention

IO-aware algorithms for training linear attention on GPUs, analogous to FlashAttention for softmax.

  • LA-1: Tiling + accumulation to avoid large intermediates
  • LA-2: Separates intra-block (causal) and inter-block components
  • Near hardware-peak throughput for linear attention training

Persistent Limitations

  • Associative recall remains challenging — fixed-size state compresses all past K-V associations
  • Precise token retrieval — copying, exact retrieval, multi-hop reasoning favor softmax attention
  • Softmax can place arbitrarily sharp probability mass on individual tokens
  • These gaps motivate hybrid architectures

RWKV: Reinventing RNNs

RWKV-4 (2023)

  • Weighted Key Value with exponential distance decay
  • Token shift for local context
  • Parallel scan (training) + simple recurrence (inference)

RWKV-5 Eagle (2024)

  • Multi-headed matrix-valued states
  • Learned matrix decay per head

RWKV-6 Finch (2024)

  • Data-dependent decay rates (like Mamba's selectivity)
  • Dynamic context retention decisions
  • Trained up to 14B params — comparable to Llama 2 13B

RWKV-7 Goose (2025)

  • Improved state management + in-context learning
  • At 7B: competitive with transformers on MMLU, HellaSwag, ARC
  • Constant-memory inference maintained
  • Fully open-source, Apache 2.0

xLSTM, Griffin & HGRN

xLSTM (Hochreiter, 2024)

  • sLSTM: Exponential gating for larger dynamic range
  • mLSTM: Matrix-valued memory (d×d) with outer-product updates
  • C_t = f_t C_{t-1} + i_t (v_t k_t¹)
  • Competitive at 1.3B but limited adoption due to sequential backprop in sLSTM

Griffin / Hawk (DeepMind, 2024)

  • Hawk: Single-head gated linear recurrence
  • Griffin: Multi-head recurrence + sliding window attention
  • At 7B: matched transformer quality
  • Recurrent state generalizes to longer sequences better than positional encodings

HGRN / HGRN2

  • Hierarchical gated recurrence at multiple timescales
  • Different forget-gate biases create fast → slow dimensions
  • HGRN2 adds outer-product state expansion
  • Competitive with Mamba at 1.3–3B
  • Notable for simplicity

RetNet: Retentive Networks

Retention Mechanism

Retention(Q,K,V)_n = Σ_{m≤n} γ^{n-m} (Q_n K_m¹) V_m

γ = per-head learned decay rate

Multi-scale retention uses different decay rates per head — some focus locally, others maintain long-range information.

Three Computation Modes

  • Parallel: Matrix operation for training
  • Recurrent: Linear RNN for O(1) inference
  • Chunk-wise: Parallel within chunks, recurrent between — throughput-oriented inference

Results at 6.7B

  • Quality comparable to transformers
  • 8.4x faster decoding
  • 3.4x lower memory during inference
  • Microsoft Research, 2023

Significance

RetNet showed that exponential decay + multi-scale heads could approximate attention quality while enabling all three compute paradigms from a single formulation.

Hybrid Architectures

Jamba (AI21 Labs, 2024)

  • First production-scale hybrid: 7:1 Mamba-to-attention ratio + MoE
  • 52B params (12B active), 256K context
  • Fits on single 80GB GPU for inference
  • Matched Mixtral 8x7B and Llama 2 70B

Zamba (Zyphra, 2024)

  • Shared attention layers interleaved among Mamba blocks
  • Weight sharing reduces param count
  • Even 1–2 shared attention sets significantly improve recall

Samba (Microsoft, 2024)

  • Mamba + sliding window attention (not full attention)
  • Avoids quadratic cost entirely
  • SSM for long-range, local attention for precision

RecurrentGemma (DeepMind, 2024)

  • Griffin-based: gated recurrence + local attention
  • 2B and 9B models, trained on substantially more data
  • Matched Gemma 2B quality

Test-Time Computation Models

Mixture of Depths

Raposo et al., DeepMind 2024

  • Tokens skip layers via learned router
  • Capacity budget per layer (e.g. 50%)
  • Easy tokens (articles, punctuation) bypass computation
  • Top-k routing applied to depth not width

Early Exit

CALM & related work

  • Prediction heads at intermediate layers
  • Confident predictions exit early
  • 2–3x speedups with minimal quality loss
  • Challenge: calibrating exit threshold

Think / Pause Tokens

Goyal et al., 2024

  • Special tokens for extra "thinking time"
  • More forward passes to refine representations
  • Benefit scales with pause count
  • Orthogonal to architecture choice

Convergence & Future Outlook

The Convergence Hypothesis

Many architectures are converging mathematically:

  • Mamba-2 SSD: selective SSMs = structured attention
  • GLA, RetNet, RWKV: gated linear recurrences with matrix-valued states
  • mLSTM closely related to linear attention

The field may be discovering a common computational primitive — a gated linear recurrence with input-dependent parameters.

Trajectory

The question is shifting from "SSMs vs transformers" to "what is the optimal ratio and placement of attention layers within a predominantly recurrent architecture?"

Open Questions

  • State capacity: Minimum state size to match transformer quality at scale?
  • Optimal hybrid ratios: How much attention vs recurrence? Does it change with scale?
  • Training stability: Multiplicative state updates at 100B+ scale
  • Hardware co-design: Custom hardware for scan ops (Groq LPU, FPGAs)
  • Distillation: Can pretrained transformers convert to SSM without retraining?

Scaling Gap

No pure SSM model publicly trained at 70B+ with comparable data. Jamba (52B active) is the strongest evidence that SSM-based architectures can scale to production quality.

Key Takeaways

SSMs Work

Mamba and successors achieve transformer-competitive quality at moderate scales with linear complexity, O(1) inference memory, and 5x+ decode speedups.

Hybrids Are the Sweet Spot

Selective attention (10–15% of layers) within a recurrent backbone delivers best practical trade-off — near-transformer quality at greatly reduced cost.

Ecosystem Is Maturing

Custom CUDA kernels, Triton implementations, GGUF quantization, and serving framework support are closing the infrastructure gap.

Architectures Are Converging

SSD duality reveals that SSMs, linear attention, and gated RNNs are different implementations of the same primitive — gated linear recurrence.

The Transformer Is Not Dead

Full attention remains unmatched for precise retrieval, in-context learning, and frontier-scale models. But its position as the sole viable architecture is no longer uncontested.
